# BP Technical Challenge - Frontend

## Description

This project is a technical challenge. It is designed to provide an intuitive and responsive user interface for the management of banking products and related functionalities.

## Features

- Modern and responsive user interface.
- Integration with backend APIs.
- Support for authentication and authorization.
- Reusable components with [framework utilized, e.g., React/Vue/Angular].

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/blilllly/dev-front-technical-challenge.git
   ```
2. Navigate to the project directory:
   ```
   cd <carpeta-del-proyecto>
   ```
3. Install the dependencies:
   ```
   npm install
   ```
4. Run the project in development mode:
   ```
   ng s -o
   ```

## Api

running the project in development mode will allow you to access the API endpoints provided by the interviewer. Make sure to have the backend server running and properly configured to connect with the frontend application.

## Available Scripts

- `ng s -o`: Runs the development server and opens the application in the browser.
- `npm run start:dev`: Runs the development server provided by the interviewer.
- `ng test`: Runs the tests.
- `ng test --coverage`: Runs a code coverage interface.
